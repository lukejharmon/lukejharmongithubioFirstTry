A note on links and pdfs

For some of the papers above, you will see both a link to the paper on the journals homepage and a pdf file hosted here. If you have institutional access to the journal in question, please try the "link" first; only download the pdf if that doesn't work. Journals are starting to use web hits as an index of scientific impact, so using these links helps us all.
